# Snow Simulation
Eine Schneefall-Simulation, kreiert mit Python und Pygame

## Features
- Zufällige Schneeflocken mit unterschiedlichen Größen und Geschwindigkeiten
- Objektorientierte Programmierung
- Zufällige Farbem
- Windbewegung (in Arbeit)
- FPS-Anzeige (in Arbeit)
- Einstellbare Schneemenge (in Arbeit)

## Technologien
- Pygame
- Python

## Installation
```bash
pip install -r requirements.txt
python main.py
```
